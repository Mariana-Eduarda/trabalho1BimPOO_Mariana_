## **Empresa SkyBlue**

 public class trabalhoTeste {


  public static void main(String[] args {

    System.out.println("Olá, seja bem vindo(a) à SkyBlue");

  }

}

Nesse exercício eu desenvolvi um programa simples em Java para representar o início de um sistema da empresa SkyBlue.

A ideia principal foi criar uma estrutura básica usando a classe principal (trabalhoTeste) com o método main, que é o ponto de entrada de qualquer aplicação Java.

Dentro desse método, eu utilizei um comando de saída (System.out.println) para exibir uma mensagem na tela:

"Olá, seja bem vindo(a) à SkyBlue"

Esse tipo de programa é bem inicial, mas é importante porque ajuda a entender como funciona a execução de um código em Java, além da estrutura básica de uma aplicação.

Também dá pra considerar que esse seria o começo de um sistema maior, onde futuramente poderiam ser adicionadas novas funcionalidades, como cadastro de usuários, login, ou até integração com outros módulos da empresa.
